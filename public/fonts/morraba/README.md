# Morraba font files

Drop the official Morraba `.woff2` files here:

- `Morraba-Light.woff2`   (300)
- `Morraba-Regular.woff2` (400)
- `Morraba-Medium.woff2`  (500)
- `Morraba-Bold.woff2`    (700)

Morraba is a commercial Persian font from FontIran — purchase or license
the files separately, then place them in this directory. The `@font-face`
rules in `app/globals.css` already point here.

While these files are missing, the site falls back to **Estedad**
(loaded from jsDelivr CDN).
